Product Details in Sale order


The "Product Details in Sale order"The Product Details on Sale order module is an importable module that can be easily imported into the server and Users can view the product name and quantity.

User could access Product Details on Sale order
Enable or disable product details

To navigate this functionality, Go to the Apps and click on the "Import Module" menu item.


From here, You can upload the zip file of the module and click on the Import App.


From here, You can easily add a group and then, you'll be able to see the product detail field.


After that, Users can enable or disable the product detail fields in Sale order.
